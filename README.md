# MasterKittens
MasterKittens è un card game strategico ispirato al celebre party game Exploding Kittens. Affronta 3 bot, usa carte speciali e cerca di evitare le bombe esplosive per rimanere l’ultimo sopravvissuto.

Il gioco include:
* Partite rapide contro 3 bot
* Carte speciali come Attacco, Salta, Mischia e Prevedi
* Sistema di bombe e disinnesco
* IA semplice e dinamica
* Interfaccia ottimizzata per desktop e mobile
* Sistema di vittorie salvate localmente

È collegato alla repository centrale MasterHub (https://mastersabba.github.io/MasterSabba/), piena di altri minigame.
